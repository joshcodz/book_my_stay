# book_my_stay

A Java-based hotel booking simulation project demonstrating core Java structures, object-oriented design, validation, cancellation, and concurrent booking processing.

## Repository structure

- `src/main/java/` — Java source files for the hotel booking system
- `.gitignore` — ignores compiled `.class` files and common IDE artifacts
- `README.md` — project overview and usage instructions

## Getting started

1. Compile the source:
   ```powershell
   javac src/main/java/*.java
   ```
2. Run a sample simulation:
   ```powershell
   java -cp src/main/java UseCase11ConcurrentBookingSimulation
   ```

## Notes

- Keep only source files under `src/main/java`
- Use `.gitignore` to prevent compiled classes from being tracked
